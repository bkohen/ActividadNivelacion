# ActividadNivelacion

## Analogía
Sistema de Dispositivos Inteligentes (Smart Home)

## Archivos
- dispositivo.py → Clase abstracta
- wifi.py → Funcionalidad WiFi
- bateria.py → Funcionalidad batería
- dispositivo_inteligente.py → Herencia múltiple

## Ejecución
Ejecutar:

python dispositivo_inteligente.py

## Descripción
Se implementa una clase abstracta con dos métodos obligatorios,
dos clases independientes y una clase final con herencia múltiple.
Se aplica encapsulamiento usando property y setter con validación.
